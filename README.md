## Getting Started
Setting OpenAI API key:  create `.env` set `NEXT_PUBLIC_API_KEY="sk-xxx"`

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fmggger%2Fspeech-to-text&env=NEXT_PUBLIC_API_KEY)

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
